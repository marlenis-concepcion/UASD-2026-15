# Agentes del Curso

Este folder guarda agentes y prompts reutilizables para generar contenido docente de INF-8018-C9.

## Agentes sugeridos

- `agente_diseno_unidad.md`: convierte un tema en una unidad completa con objetivos, contenidos, actividades y evaluacion.
- `agente_foros.md`: genera foros academicos, preguntas detonadoras y respuestas de seguimiento.
- `agente_tareas.md`: redacta instrucciones de tareas, criterios y formato de entrega.
- `agente_rubricas.md`: crea rubricas analiticas y listas de cotejo.
- `agente_retroalimentacion.md`: redacta comentarios docentes claros, formativos y respetuosos.
- `agente_apa7_uasd.md`: revisa formato academico, citas y referencias cuando se necesite APA 7.

## Regla de uso

Todo contenido generado debe revisarse antes de publicarse en UASDVirtual, especialmente instrucciones, criterios de evaluacion, fechas, ponderaciones y referencias.
