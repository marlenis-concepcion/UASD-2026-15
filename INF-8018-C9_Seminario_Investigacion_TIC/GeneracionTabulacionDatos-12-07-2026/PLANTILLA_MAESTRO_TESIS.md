## XV. DIFERENCIAS CLAVE: CUANTITATIVO vs CUALITATIVO vs MIXTO

### Enfoque Cuantitativo:
✓ Números, estadísticas, gráficos
✓ Responde: ¿CUÁNTO? ¿CON QUÉ FRECUENCIA?
✓ Mejor para: Medición, comparación, correlación
✗ Menos bueno para: Entendimiento profundo, contexto

### Enfoque Cualitativo:
✓ Narrativas, citas, temas
✓ Responde: ¿POR QUÉ? ¿CÓMO?
✓ Mejor para: Comprensión profunda, contexto, experiencias
✗ Menos bueno para: Generalización, medición

### Enfoque Mixto (RECOMENDADO para maestrías):
✓ Combina lo mejor de ambos
✓ Cuantitativo: Qué sucede (números)
✓ Cualitativo: Por qué sucede (narrativa)
✓ Triangulación: Valida hallazgos

## XVI. APÉNDICE: FÓRMULAS BÁSICAS

### Media (Promedio):
```
M = (Σ Xi) / n
Donde: Xi = cada valor, n = número total
```

### Desviación Estándar:
```
DE = √[Σ(Xi - M)²] / n]
Mide variabilidad alrededor de la media
```

### Correlación de Pearson:
```
r = Σ[(Xi - Mx)(Yi - My)] / √[Σ(Xi - Mx)² × Σ(Yi - My)²]
Rango: -1 a +1
|r| > 0.7 = Fuerte
0.4 < |r| < 0.7 = Moderada
|r| < 0.4 = Débil
```

### Porcentaje:
```
% = (Parte / Total) × 100
```
