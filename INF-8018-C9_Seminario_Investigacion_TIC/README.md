# INF-8018-C9 - Seminario de Investigacion en TIC

Carpeta de trabajo docente para organizar recursos, actividades, agentes y contenido generado de la asignatura.

## Datos del curso

- Plataforma: UASDVirtual
- Asignatura: Seminario de Investigacion en TIC
- Codigo: INF-8018-C9
- Docente: Marlenis Judith Concepcion Cuevas
- Unidad inicial: Unidad 1 - Construccion del anteproyecto de investigacion

## Estructura

- `00_admin`: datos administrativos, calendario, cronograma y notas internas.
- `01_programa_y_recursos`: programa de la asignatura y materiales base.
- `02_unidades`: desarrollo por unidad, guiones, contenidos y planificacion.
- `03_foros`: consignas, respuestas modelo y criterios de participacion.
- `04_tareas`: instrucciones, plantillas de entrega y ejemplos.
- `05_rubricas`: rubricas, listas de cotejo y criterios de evaluacion.
- `06_agentes`: definicion de agentes, prompts del sistema y flujos de trabajo.
- `07_contenido_generado`: borradores generados para clases, foros, tareas y retroalimentacion.
- `08_estudiantes_y_evaluacion`: seguimiento, calificaciones exportadas y observaciones.
- `09_exportaciones`: archivos listos para subir a Moodle/UASDVirtual.

## Flujo sugerido

1. Guardar los materiales originales del aula en `01_programa_y_recursos`.
2. Crear o actualizar cada unidad dentro de `02_unidades`.
3. Diseñar foros y tareas en `03_foros` y `04_tareas`.
4. Definir rubricas antes de publicar actividades.
5. Usar los agentes de `06_agentes` para generar borradores.
6. Pasar solo versiones revisadas a `09_exportaciones`.
