## 🔗 REFERENCIAS RÁPIDAS

### Formato de Citaciones Comunes

**Un autor:**
```
González (2020) afirma que... (González, 2020)
```

**Dos autores:**
```
García y Méndez (2017) plantean... (García y Méndez, 2017)
```

**Tres o más:**
```
Morrison et al. (2015) demuestran... (Morrison et al., 2015)
```

**Cita textual corta (< 40 palabras):**
```
Según Kaplan y Szapu (2019), "el texto exacto va aquí" (p. 111).
```

**Cita textual larga (≥ 40 palabras):**
```
Kaplan y Szapu (2019):
    Texto completo con sangría izquierda de 1.27 cm sin
    comillas. Punto ANTES del paréntesis. (p. 111)
```

---

### Formato de Referencias Comunes

**Libro:**
```
Apellido, A. (Año). Título en cursiva. Editorial.
```

**Artículo científico:**
```
Apellido, A. (Año). Título artículo. Revista en Cursiva, volumen(número), pp-pp. URL
```

**Página web:**
```
Institución. (Año). Título. URL
```

**Tesis:**
```
Apellido, A. (Año). Título [Tesis de grado, Institución]. Archivo digital. URL
```

## 💡 TIPS RÁPIDOS

✓ **Márgenes:** Usar vista previa antes de imprimir  
✓ **Sangría francesa:** Hacerla automáticamente en Word  
✓ **Orden alfabético:** Verificar manualmente referencias  
✓ **Cursivas:** Usar tecla Ctrl+I, no subrayado  
✓ **Espaciado:** Revisar no haya espacios extra entre párrafos  
✓ **Fechas:** Formato español (23 de noviembre, 2020)  
✓ **"y" vs "&":** Usar siempre "y" en español  
✓ **URLs:** Verificar vigencia y accesibilidad
