# Casos de docentes del taller

Esta carpeta recibirá una subcarpeta independiente por cada programa o asignatura trabajada.

No se debe guardar ningún resultado de una clase dentro de `../02_Agentes_Rediseno_Curricular/`. Los agentes permanecen congelados y funcionan únicamente como plantillas de consulta.

Un caso existente solo puede modificarse cuando la propietaria del proyecto lo solicite explícitamente.
