## 🚀 CÓMO USAR

### Opción 1: Claude Code (Más fácil)
```bash
1. Abrir carpeta AGENTES en VS Code
2. Instalar extensión Claude Code
3. Escribir: /crear-procedimiento [nombre]
4. ¡Listo!
```

### Opción 2: GitHub Copilot
```bash
1. Abrir archivo del skill
2. En Copilot Chat: "Ayúdame siguiendo este procedimiento"
3. Proporcionar contexto
4. Copilot ejecuta los pasos
```

### Opción 3: ChatGPT / Claude.ai
```bash
1. Copiar contenido del skill
2. Pegar como contexto/system prompt
3. Hacer solicitud específica
4. LLM ejecuta el procedimiento
```

### Opción 4: API (Programático)
```python
with open("crear-procedimiento.md") as f:
    instructions = f.read()
    
response = llm.chat(
    system=instructions,
    user="Crear procedimiento: [nombre]"
)
```
