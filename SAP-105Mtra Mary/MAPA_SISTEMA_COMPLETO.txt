Para CADA TESIS a validar:

┌─ PASO 1: REVISIÓN PRELIMINAR RÁPIDA (5-10 min)
│  ├─ Usar: CHECKLIST_RAPIDA_APA_7ma.md
│  ├─ Evaluar: 6 secciones principales
│  ├─ Marcar: ✓ OK / ⚠ Observaciones / ✗ Crítico
│  └─ Resultado: Decidir si continuar con validación completa
│
├─ PASO 2: VALIDACIÓN DETALLADA COMPLETA (30-60 min)
│  ├─ Usar: AGENTE_VALIDACION_NORMAS_APA_7ma.md
│  ├─ Revisar: 7 secciones exhaustivamente
│  ├─ Documentar: Todos los hallazgos
│  ├─ Generar: Reporte con plantilla incluida
│  └─ Resultado: % de conformidad (90-100% / 75-89% / <75%)
│
├─ PASO 3: DECISIÓN Y COMUNICACIÓN
│  ├─ 90-100% CONFORME → APROBAR FINAL
│  ├─ 75-89% OBSERVACIONES → CORRECCIONES MENORES (3-5 días)
│  └─ <75% NO CONFORME → CORRECCIONES IMPORTANTES (7-10 días)
│
└─ PASO 4: RE-VALIDACIÓN (SI APLICA)
   ├─ Usar: CHECKLIST_RAPIDA (para secciones corregidas)
   ├─ Verificar: Solo cambios realizados
   └─ Aprobar: Cuando esté 100% conforme
