## Criterios de publicación

- Se conservan los PDF académicos, leyes, reglamentos y programas de asignaturas.
- No se publican correos, listas privadas, archivos `.txt`, capturas `.png`, invitaciones de calendario ni comprobantes de boletos.
- No se incluyen rutas privadas de computadoras, dependencias locales, metadatos de Git ni archivos temporales.
- La IA apoya el trabajo académico, pero cada resultado requiere revisión y validación humana.
