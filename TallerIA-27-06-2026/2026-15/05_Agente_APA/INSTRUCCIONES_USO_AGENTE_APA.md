## 🚀 FLUJO DE TRABAJO RECOMENDADO

### PASO 1: REVISIÓN PRELIMINAR RÁPIDA ⚡
**Duración:** 5-10 minutos  
**Herramienta:** CHECKLIST_RAPIDA_APA_7ma.md

```
1. Abrir checklist rápida
2. Evaluar 6 secciones principales
3. Marcar estado: ✓ OK | ⚠ Observaciones | ✗ Crítico
4. Decidir si proceder con validación detallada
```

**Resultado esperado:**
- Documento conforme (✓): Proceder a validación detallada
- Documento con observaciones (⚠): Hacer correcciones menores, luego validación detallada
- Documento no conforme (✗): Correcciones importantes necesarias, luego validación detallada

---

### PASO 2: VALIDACIÓN DETALLADA COMPLETA 🔍
**Duración:** 30-60 minutos  
**Herramienta:** AGENTE_VALIDACION_NORMAS_APA_7ma.md

#### FASE 2.1: PREPARACIÓN (5 min)
```
□ Tener documento a validar (Word, PDF, etc.)
□ Tener acceso a referencia de normas APA
□ Imprimir o tener visible el agente completo
□ Preparar plantilla de reporte (última página del agente)
```

#### FASE 2.2: REVISIÓN POR SECCIÓN (50-55 min)

**Sección 1: Formato General (5 min)**
```
Revisar:
- Márgenes exactos (2.54 cm)
- Font y tamaño
- Interlineado (2.0)
- Alineación (izquierda sin justificar)
- Numeración de páginas
- Abreviaturas utilizadas

Acción si hay problemas:
- Documentar en reporte
- Especificar línea de corrección
```

**Sección 2: Estructura del Documento (5 min)**
```
Revisar:
- Presencia de todos elementos requeridos
- Portada: datos completos
- Resumen: extensión y formato
- Palabras clave: cantidad y formato
- Orden lógico de elementos

Acción si hay problemas:
- Documentar elemento faltante
- Especificar corrección
```

**Sección 3: Niveles de Títulos (5 min)**
```
Revisar:
- Máximo 5 niveles
- Jerarquía consistente
- Formato de cada nivel
- Mayúsculas según español
- No saltos de niveles

Acción si hay problemas:
- Documentar título problemático
- Especificar formato correcto
```

**Sección 4: Citaciones (15 min - CRÍTICA)**
```
Proceso:
1. Seleccionar 20 citas del documento (variadas)
2. Revisar cada una contra criterios:
   - Formato narrativo: Autor (Año) correcto
   - Formato parentético: (Autor, Año) correcto
   - Textuales: comillas, página (p. o pp.)
   - Parafraseo: sin comillas, autor y año
   - Múltiples autores: uso correcto de "y"
   - Número de autores: formato correcto (et al., etc.)
3. Verificar coincidencia con referencias

Errores comunes a buscar:
- Falta de página en citas textuales
- Uso de "&" en vez de "y"
- Citas sin referencia correspondiente
- Referencias no citadas en texto
- Formato inconsistente

Acción si hay problemas:
- Documentar cita específica
- Línea aproximada en documento
- Formato correcto según APA
```

**Sección 5: Referencias Bibliográficas (15 min - CRÍTICA)**
```
Proceso:
1. Verificar sección "Referencias":
   - Título centrado, negrita
   - Orden alfabético
   - Sangría francesa
   - Interlineado 2.0

2. Seleccionar 10-15 referencias (variadas):
   - Libros
   - Artículos científicos
   - Páginas web
   - Otras fuentes presentes

3. Revisar cada una:
   - Formato correcto por tipo
   - Puntuación exacta
   - Cursivas en lugares correctos
   - Autor(es) correctamente listados
   - Fecha presente
   - URL/DOI vigentes (si aplica)

4. Verificar:
   - Cada referencia está citada en texto
   - Cada cita en texto está en referencias
   - Consistencia de formato similar

Errores comunes a buscar:
- Orden alfabético incorrecto
- Sangría francesa ausente o mal aplicada
- Puntuación inconsistente
- Título no en cursiva
- Fechas en formato incorrecto
- URLs rotas
- Referencias sin citar

Acción si hay problemas:
- Documentar referencia exacta
- Especificar error y corrección
```

**Sección 6: Tablas y Figuras (3 min)**
```
Si hay tablas:
□ Numeradas consecutivamente
□ Mencionadas en texto
□ Etiqueta "Tabla 1" - negrita
□ Título en cursiva
□ Nota presente
□ Contenido claro, legible
□ Crédito de fuente (si no es propia)

Si hay figuras:
□ Numeradas consecutivamente
□ Mencionadas en texto
□ Etiqueta "Figura 1" - negrita
□ Título en cursiva
□ Alta resolución
□ Elementos etiquetados
□ Fuente sin serifa (8-14 pts)
□ Nota presente
□ Crédito de fuente (si no es propia)

Acción si hay problemas:
- Documentar tabla/figura específica
- Especificar elemento faltante
```

**Sección 7: Adaptaciones al Español (2 min)**
```
Revisar:
□ Uso de "y" (NO "&" en narrativas)
□ Mayúsculas: solo inicio + nombres propios
□ Fechas: formato español (día mes, año)
□ Números ordinales: (3.ª ed.)
□ Puntuación y ortografía RAE

Acción si hay problemas:
- Documentar ejemplos específicos
- Especificar corrección
```

---

### PASO 3: GENERACIÓN DE REPORTE (5-10 min)
**Herramienta:** Plantilla de reporte en agente completo

```
1. Completar sección "Resumen Ejecutivo"
2. Calcular porcentaje de cumplimiento:
   - Secciones conformes / Total secciones × 100
   - 90-100%: CONFORME
   - 75-89%: CONFORME CON OBSERVACIONES
   - < 75%: NO CONFORME

3. Listar hallazgos por prioridad:
   - CRÍTICOS: deben corregirse
   - IMPORTANTES: se recomienda corregir
   - MENORES: considerar para mejora

4. Proporcionar recomendaciones específicas

5. Especificar correcciones requeridas con exactitud
```

---

### PASO 4: COMUNICACIÓN Y SEGUIMIENTO (Variable)
**Tiempo:** Depende del estado del documento

```
Si CONFORME (90-100%):
→ Documentar aprobación
→ Documento listo para entrega
→ Archivar reporte como evidencia

Si CONFORME CON OBSERVACIONES (75-89%):
→ Enviar reporte a autor(a)
→ Indicar correcciones menores
→ Agendar re-validación (revisión #2)
→ Plazo: 3-5 días hábiles

Si NO CONFORME (< 75%):
→ Enviar reporte detallado a autor(a)
→ Marcar como "REQUIERE CORRECCIONES SIGNIFICATIVAS"
→ Programa de correcciones con hitos
→ Agendar re-validación (revisión #2)
→ Plazo: 7-10 días hábiles
```

---

## 🔄 PROCESO DE RE-VALIDACIÓN (Revisión #2, #3, etc.)

Cuando el autor(a) envíe correcciones:

```
1. Revisar SOLO las secciones corregidas
2. Usar CHECKLIST_RAPIDA_APA_7ma.md para validación rápida
3. Si hay problemas residuales:
   → Usar AGENTE_VALIDACION_NORMAS_APA_7ma.md para esa sección
4. Generar reporte actualizado
5. Indicar estado: "Conforme" o si hay más correcciones
6. Máximo 3 ciclos de revisión
   → Si 3ª revisión aún tiene problemas, escalara supervisor
```

## ⚠️ ERRORES MÁS COMUNES ENCONTRADOS

### En Formato
- ✗ Márgenes incorrectos (especialmente derecho e inferior)
- ✗ Interlineado 1.5 en lugar de 2.0
- ✗ Espacios extra entre párrafos
- ✗ Fuentes inconsistentes

### En Citaciones
- ✗ Páginas faltantes en citas textuales
- ✗ Uso de "&" en lugar de "y"
- ✗ Citas sin referencia correspondiente
- ✗ Formato narrativo y parentético mezclados

### En Referencias
- ✗ Orden alfabético incorrecto
- ✗ Sangría francesa ausente
- ✗ Títulos de libros/artículos sin cursiva
- ✗ Fechas en formato incorrecto
- ✗ Puntuación inconsistente
- ✗ URLs rotas o actualizadas

### En Tablas/Figuras
- ✗ No mencionadas en texto
- ✗ Títulos no en cursiva
- ✗ Notas ausentes
- ✗ Crédito de fuente faltante

### En Español
- ✗ Mayúsculas en títulos (title case)
- ✗ Uso de "&" en narrativas
- ✗ Meses con mayúscula inicial

## 📞 PREGUNTAS FRECUENTES (FAQ)

### P: ¿Cuánto tiempo tarda la validación completa?
**R:** 30-60 minutos para un documento de 50-80 páginas, dependiendo de:
- Calidad inicial del documento
- Número de citas y referencias
- Cantidad de tablas/figuras
- Complejidad del contenido

### P: ¿Qué si el documento está en PDF?
**R:** 
- Si es editable: trabajar directamente
- Si es no editable: solicitar versión Word
- Validación es más difícil en PDF

### P: ¿Qué pasa si encuentra muchos errores?
**R:** 
- Documentar todos exhaustivamente
- Marcar como "NO CONFORME"
- Proporcionar guía clara de correcciones
- Hacer re-validación después de correcciones

### P: ¿Puede validar solo una sección?
**R:** 
- Sí, para re-validaciones
- Usar CHECKLIST_RAPIDA_APA_7ma.md
- Enfocarse en sección modificada
- Verificar no crear inconsistencias

### P: ¿Qué si documento está casi OK?
**R:** 
- Si está 75-89%: CONFORME CON OBSERVACIONES
- Listar correcciones específicas
- Agendar re-validación
- No hacer cambios por el revisor (responsabilidad del autor)

### P: ¿Se pueden hacer cambios directo al documento?
**R:** 
**NO.** El revisor:
- Solo identifica problemas
- Documenta hallazgos
- El autor(a) hace correcciones
- Revisor valida cambios

### P: ¿Hay herramientas automáticas APA?
**R:** 
- Word tiene verificador de APA (limitado)
- Zotero/Mendeley ayudan con referencias
- Este agente es manual pero completo
- Combinar herramientas es más seguro
